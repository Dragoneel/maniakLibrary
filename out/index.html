Introduction

Maniak Library est une application web pour gérer les emprunts de livres.

L'utilisateur "normal" peut :

  • Créer un compte
  • Louer un livre
  • Rendre un livre

L'utilisateur "admin" peut :

  • Créer / modifier / supprimer un livre
  • Ajouter un autre administrateur
  • Visualiser les livres emprunter par les utilisateurs

Quand un utilisateur "normal" devient "admin" il rend tous les livres qu'il a déja emprunter

Pour que ça soit plus réaliste, on peut implementer un systeme de token, chaque livre emprunter aurra un identifiant unique, l'utilisateur récupère le livre à l'aide de ce dernier. Coté administrateur, on vérifie le token à l'aide d'un scanner ( qrcode par exemple ).
